# Red-Team Data Publication Policy

Public E5/E6 artifacts must be sanitized, non-operational, and suitable for defensive evaluation.

Allowed public content:

- placeholders such as `[restricted instructions]`
- abstract categories
- simulated refusal/helpfulness traces
- aggregate metrics

Restricted content:

- raw harmful prompt corpora
- operational exploit instructions
- weaponized payloads
- secrets, tokens, or private data

SV-Bench v0.1 does not publish offensive or red-team corpora.
